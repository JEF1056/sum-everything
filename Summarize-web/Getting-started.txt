** Gettin Started **
What?
	This gives simple instructions on how to start the server.

Prereq:
	-Python3
	-Flask
	-Web browser

1.) Navigate into 'Summarize-web' using command line

	[Windows] 
		> set FLASK_APP=sumWeb
		> set FLASK_ENV=development
		> flask run

	[Bash]
		$ export FLASK_APP=sumWeb
		$ export FLASK_ENV=development
		$ flask run

2.) Place IP given into web browser:

	Ex) Go to 'http://111.0.0.0:5000/' in:
	 * Serving Flask app 'sumWeb' (lazy loading)
	 * Environment: development
	 * Debug mode: on
	 * Restarting with stat
	 * Debugger is active!
	 * Debugger PIN: 106-779-090
	 * Running on http://111.0.0.0:5000/ (Press CTRL+C to quit)


** Files **

- __init__.py: Flask initialization file (where it starts)

- /static/styles: Where css files are stored

- /templates: Where templates [html for blueprints sometimes] are stored
